# Calculator

A small web-based calculator demo. Click the buttons or use the keyboard to enter expressions and evaluate them.

## Usage

- Open [index.html](index.html) in a browser.
- Click the buttons or type using your keyboard. Press Enter to evaluate, Backspace to delete, and Esc to clear.

This repository contains a simple, approachable implementation intended for learning and demonstration.
